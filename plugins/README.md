## Plugin 一括配置用 ZIP について

以下の Google Drive リンクから ZIP ファイルをダウンロードし、  
解凍後の内容を **サーバー側の `plugins/` フォルダ** に配置してください。

🔗 ダウンロードリンク  
https://drive.google.com/file/d/1RxoGFsyLtCN3xz7bXgKuATiep9N2LxX9/view?usp=drive_link

> ⚠ **注意**  
> 本 ZIP は **サーバー専用** です。  
> クライアント（個人PC）の `mods/` フォルダには **入れないでください**。

## Plugin の取り扱いに関する注意事項（免責事項）

### 開発・テスト目的の配布
本 Google Drive リンクは、  
**本サーバー開発チーム内での環境同期およびデバッグを目的とした一時的な配布物**です。  
一般公開や恒久的な配布を意図したものではありません。

### 権利の帰属
本 ZIP に含まれる **すべての Plugin の著作権および知的財産権は、各 Plugin 制作者に帰属**します。

### 再配布について
本プロジェクトは **Plugin の再配布を目的としたものではありません**。  
開発終了時、または Plugin 制作者からの要請があった場合、  
**本リンクは速やかに削除されます。**

### バージョンについて
本 ZIP に含まれる Plugin は、  
**検証時点で動作確認済みのバージョンを基準に構成されています**。  
必ずしも各 Plugin の最新版ではありません。

### 公式ソースの利用推奨
可能な限り、以下の公式配布元から  
**各 Plugin を個別にダウンロード・管理することを推奨**します。

- Modrinth  
- 各 Plugin の公式配布ページ
